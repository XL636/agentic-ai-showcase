We build agents that ship — not demos.

The RDCLAW (睿动AI) team designs and ships enterprise-grade agentic AI: a production desktop platform, a self-developed agent engine, proactive scheduled agents, and IM-native delivery into Feishu and DingTalk.

Windows v1.5.2 in production macOS (Apple Silicon) v0.1.10 10+ model providers 30+ built-in skills MCP support Feishu & DingTalk integration

Flagship: RDCLAW (睿动Claw) — enterprise agent desktop platform

Autonomous agent loop

Goal in, result out: agents plan, call tools (files, browser, shell, MCP), observe, and self-correct — no rigid pipelines, no hard-coded step limits.

Two-layer resilience

A deterministic engine with provider fallback chains and safe-mode degradation underneath a reasoning loop that routes around failures instead of giving up.

Proactive scheduled agents

Plain-language cron: “every weekday at 9am, summarize and send to me.” The agent wakes itself, works, and delivers into Feishu / DingTalk.

Multi-agent personas

Multiple independent agents per user, each with its own personality, memory, and tool permissions.

Visible guardrails

Three-tier command execution safety, OS-native key encryption (DPAPI), sandbox isolation, allowlists and command gating for group chats.

Open extensibility

Built on the OpenClaw open-source agent framework; capabilities attach through MCP and a plugin interface, not core hacks.
